# Task Manager App

エレガントなタスク管理Webアプリケーション

## 機能

- **Google OAuth認証** - Manus OAuthによるセキュアなユーザー認証
- **タスクCRUD操作** - タスクの作成、閲覧、編集、削除
- **期限設定** - 各タスクに期限日時を設定
- **優先度管理** - タスクに優先度(高・中・低)を設定し、フィルタリング
- **ステータス管理** - 未完了・完了のステータス切り替え
- **レスポンシブデザイン** - モバイル・タブレット・デスクトップ対応

## 技術スタック

- **フロントエンド**: React 19, Tailwind CSS 4, shadcn/ui
- **バックエンド**: Express 4, tRPC 11
- **データベース**: MySQL (Drizzle ORM)
- **認証**: Manus OAuth
- **テスト**: Vitest

## デザイン

エレガントなパープルトーンのカラーパレットを採用し、洗練されたユーザーインターフェースを実現しています。

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# データベースマイグレーション
pnpm db:push

# 開発サーバー起動
pnpm dev

# テスト実行
pnpm test
```

## 環境変数

プロジェクトには以下の環境変数が自動的に設定されます:

- `DATABASE_URL` - データベース接続文字列
- `JWT_SECRET` - セッション署名用シークレット
- `OAUTH_SERVER_URL` - OAuth認証サーバーURL
- その他のManus統合用環境変数
